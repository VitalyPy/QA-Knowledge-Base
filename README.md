# QA-Knowledge-Base
Данный репозиторий представляет собой личную базу знаний по ручному и автоматизированному тестированию. Здесь в структурированном виде собраны основные концепции, чек-листы и заметки по ключевым областям QA для быстрого поиска и повторения. Структура репозитория заложит основу для будущих примеров тестового кода и скриптов.
